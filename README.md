# AKI暗記 — 用語暗記アプリ

仕事（航空運航支援）の用語の **定義を一言一句** 覚えるための、iPhone 向け学習アプリ。
×〇クイズではなく「定義をそのまま確認」する方式。間隔復習で出題し、進捗を見られます。

- **公開 URL**: https://hyvamps0812-hue.github.io/aki-anki/
- **本体**: [index.html](index.html) … これ 1 ファイルで完結（HTML+CSS+JS、ビルド不要）
- **データ**: すべて端末内（ブラウザの localStorage）に保存。サーバー・アカウント・ログインなし。
- 設計の正: [docs/specs/2026-06-08-term-memo-app-design.md](docs/specs/2026-06-08-term-memo-app-design.md)（Claude Design の SPEC.md / aki.css はローカル `_design_src/` に保管・リポジトリ対象外）

## iPhone に入れる

1. iPhone の **Safari** で https://hyvamps0812-hue.github.io/aki-anki/ を開く。
2. 共有ボタン → **「ホーム画面に追加」**。
3. ホーム画面の「AKI暗記」から全画面で起動。2 回目以降はオフラインでも開けます（Service Worker）。

> 学習データは「開いた URL」に紐づいて端末内に保存されます。この URL で使い続けてください。
> パソコン（Windows）でも、`index.html` をダブルクリックすれば Chrome/Edge で動作確認できます。

## 使い方

- **学習**: 用語を見て → タップで定義＋補足を全文表示 →「覚えてた／あいまい／忘れてた」を押すだけ。次に出す日が自動で決まります。
  - 上部の **読むだけ / 入力** で切替。「入力」は、自分で定義を書いて（🎤音声入力も可）答え合わせ → **書けなかった部分がハイライト** されます。
- **一覧**: 全用語を辞書のように閲覧・検索・カテゴリ絞り込み。行をタップで編集・削除。
- **追加**: 用語・定義・補足・カテゴリ・要点キーワードを登録（各欄ともキーボードの🎤で音声入力可）。
- **進捗**: 定着率・カテゴリ別・苦手ランキング・連続学習日数・復習待ち数。
- **設定（右上の歯車）**: バックアップ書き出し/読み込み、テーマ（端末/ライト/ダーク）、AI、サンプル投入/全削除。

## バックアップ（大事）

端末内保存は、Safari のデータ消去や機種変更で **消えることがあります**。
設定 →「書き出し」で全データを JSON ファイルに保存できます（月 1 回など定期的に推奨）。戻すときは「読み込み」。

## AI に相談（任意）

進捗と「入力モードの間違い記録」を読んで、弱点傾向と次の一手を AI が日本語で提案します。本体は AI なしで全機能動きます。

1. 設定 → AI →「有効化」ON → **自分の AI 利用キー** を入力。
2. プロバイダ（OpenAI / Anthropic / Google）とモデルを選択。
3. 進捗画面の「AI に相談する」で実行。

- 使った瞬間だけ、本人の学習データが AI 提供元へ送られます（設定にも明記）。少額の従量課金あり。
- キーは端末内のみに保存し、**バックアップの書き出しには含めません**。
- 提供元によってはブラウザからの直接呼び出し（CORS）を制限する場合があります。エラー時はキー・モデル名・通信状況をご確認ください。

## 技術メモ

- React + Babel のデザイン試作を、**素の JavaScript 単一ファイル**に移植（オフライン・起動速度のため Babel を排除）。
- デザインは `aki.css` を忠実に埋め込み（Cockpit テーマ）。ロジックは SPEC.md §6（間隔復習）/ §7（ハイライト）/ §9（バックアップ）に準拠。実日付・空状態・追加/編集/削除・バリデーション・LCS フォールバック・attempts 記録・ライトテーマ・連続日数の実計算・PWA メタ・AI 呼び出しを実装。
- PWA: `manifest.json` + `apple-touch-icon.png` / `icon-512.png` + `sw.js`（stale-while-revalidate。同一オリジンと Google Fonts のみキャッシュ）。
- 更新の反映: このリポジトリの `main` に push すると GitHub Pages が自動再配信。iPhone 側はアプリを開き直すと次回起動時に新版へ。
